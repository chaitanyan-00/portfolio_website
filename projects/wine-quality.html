Wine Quality Prediction

Machine learning model to predict wine quality based on chemical properties

View on GitHub

Project Overview

This project involved developing a machine learning model to predict wine quality based on various chemical properties. The goal was to explore different algorithms and feature selection techniques to achieve the best prediction accuracy.

Python Pandas Scikit-learn Matplotlib Jupyter Notebook

Technical Implementation

  • Collected and cleaned dataset of wine characteristics from UCI Machine Learning Repository
  • Performed exploratory data analysis to identify key features affecting wine quality
  • Developed and compared multiple regression and classification models
  • Optimized model performance using hyperparameter tuning
  • Achieved 85% accuracy in quality classification

Challenges & Solutions

Imbalanced Dataset

The dataset had significantly more average-quality wines than excellent or poor ones.

Solution: Applied SMOTE (Synthetic Minority Over-sampling Technique) to balance the classes.

Feature Selection

Determining which chemical properties had the most impact on quality was challenging.

Solution: Used recursive feature elimination and correlation analysis to identify the most significant features.
